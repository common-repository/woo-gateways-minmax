Set minimum and maximum amount for payment gateways in WooCommerce.


== Description ==
Some online store manager would like to deactivate one or more payment methods by order total amount. This plugin is what they need. For examlpe you may want to deactivate "cash on delivery" for orders less than 100$, Or you may want to deactivate "PayPal" for orders more then 500$. Try this plugin!


== Installation ==
 
THE YOLO WAY

1. In your WordPress Admin, go to the Add New Plugins page
2. Search for: Gateways MinMax for WooCommerce
3. It should be the first result. Click the Install link.
4. Click the Activate link.


THE OLD-BUT-GOLD WAY

1. Upload the plugin to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
